## Variants

### `Preinstalled`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Preinstalled"}} /></pre>

Harness is already available in the prepared environment.

### `ImageProvided`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "ImageProvided"}} /></pre>

Environment image already contains the harness and dependencies.

### `PipUv`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "PipUv"}} /></pre>

Adapter may install a Python package with pip or uv.

### `Npm`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Npm"}} /></pre>

Adapter may install a Node package.

### `Source`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Source"}} /></pre>

Adapter may install from source.

### `Service`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "Service"}} /></pre>

Adapter connects to an already-running service.

### `NativePlugin`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "NativePlugin"}} /></pre>

Adapter is installed through a harness-native plugin manager.
