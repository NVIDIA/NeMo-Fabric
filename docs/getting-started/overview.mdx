NeMo Fabric standardizes how applications configure, launch, invoke, and collect
artifacts from agent harnesses. The Python SDK (`nemo_fabric.FabricClient`) is the
primary API for consumers (Platform, Harbor/Gym, local users): resolve an agent's
config, plan it, diagnose it, and run it through a harness — with normalized
results and no harness-specific code in the caller.

## Install

```bash
pip install -e .        # installs the SDK with the native binding (maturin)
```

## Quickstart

Run from an agent package:

```python
import asyncio
from nemo_fabric import FabricClient

async def main():
    async with FabricClient() as client:
        result = await client.run(
            "examples/code-review-agent",
            profiles=["hermes_sdk"],
            input="review this diff",
        )
    print(result.status)

asyncio.run(main())
```

Or build the Fabric slice from a typed config instead of materializing an agent
package:

```python
from nemo_fabric import FabricConfig

config = FabricConfig.from_mapping(
    {
        "schema_version": "fabric.agent/v1alpha1",
        "metadata": {"name": "code-review-agent"},
        "harness": {"adapter_id": "nvidia.fabric.hermes.sdk"},
        "models": {"default": {"provider": "nvidia", "model": "nvidia/nemotron-3-nano-30b-a3b"}},
    }
)

result = await client.run(
    config,
    base_dir="examples/code-review-agent",
    input="review this diff",
)
```

The full method-by-method reference is under **Reference → APIs → Python Library
Reference**, generated from the SDK docstrings.
