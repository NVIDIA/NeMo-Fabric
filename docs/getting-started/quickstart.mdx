Use this quickstart to build NeMo Fabric from a source checkout and run the
maintained code-review example with Hermes Agent. The time required depends on
your local build speed.

## Prerequisites

Before you begin, complete the
[source installation prerequisites](/getting-started/install#source-installation-method).
You need the following tools and source checkout:

- A local checkout of the NeMo Fabric repository.
- Rust and Cargo.
- Python 3.11–3.13.
- `uv`.
- An `NVIDIA_API_KEY` for NVIDIA-hosted model access.

Run the commands in this quickstart from the root of the source checkout.

## Quickstart Steps

Complete the following steps in order from the repository root.

### Build NeMo Fabric

Install `just` 1.50.0+ if it is not already available.

```bash
cargo install just --locked
```

Refer to the [official installation guide](https://just.systems/man/en/installation.html)
for more details.

Ensure that the Cargo bin directory is in your `PATH`:

```bash
export PATH="$HOME/.cargo/bin:$PATH"
```

Create a Python virtual environment and activate it (replace `3.13` with your preferred Python version):

```bash
uv venv -p 3.13 --seed .venv
source .venv/bin/activate
```

From the root of the source checkout, build and install the Python SDK and its
native binding in `.venv`, build the CLI, and create the local wheels:

```bash
just build-all
just wheels
```

### Install Hermes Agent

Create a separate environment for Hermes Agent and install the harness and its
NeMo Fabric adapter from the local wheels:

```bash
python3 -m venv .tmp/hermes-venv
.tmp/hermes-venv/bin/python -m pip install --find-links dist "nemo-fabric[hermes]"
```

Set the model API key and select the Hermes Agent environment:

```bash
export NVIDIA_API_KEY=...
export ADAPTER_PYTHON="$PWD/.tmp/hermes-venv/bin/python"
```

### Run the Example

Run the maintained code-review example through the Python SDK:

```bash
.venv/bin/python -m examples.code_review_agent \
  --input "Reply with exactly: fabric works"
```

The quickstart is complete when the command finishes successfully and prints
`fabric works` in the agent response.
