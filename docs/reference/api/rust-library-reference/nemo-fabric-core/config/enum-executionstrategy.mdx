How the selected adapter executes harness work for one Fabric runtime.

## Variants

### `ProcessPerInvocation`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "ProcessPerInvocation"}} /></pre>

Launch a fresh adapter process for each invocation.

### `PersistentLocalHost`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "PersistentLocalHost"}} /></pre>

Start one adapter-owned local host and reuse it for the runtime.

### `RemoteService`

<pre className="rust-signature"><code dangerouslySetInnerHTML={{"__html": "RemoteService"}} /></pre>

Allocate or connect to an adapter-owned remote harness service.
